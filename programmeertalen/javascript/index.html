Javascript

Wat is Javascript?

JavaScript kan HTML en CSS programma’s updaten en veranderen. Het kan berekeningen maken, data manipuleren en valideren. De programmering bepaalt of dat dit alles automatisch gebeurt of getriggerd wordt door een handeling van de bezoeker. Dit laatste wil zeggen, een actie van de bezoeker geeft de aanzet voor de uitvoering van het script en veroorzaakt een ‘gebeurtenis’ op de webpagina.

Meer weten over Java? Klik hier.

Wat doen wij met Javascript?

Op deze opleiding krijg je wat later in het leerjaar ook te maken met javascript. Dit is vooral om websites wat meer functies te geven. Bijvoorbeeld een onclick functie geven aan een button waardoor je je html en css manipuleert.

Welke bedrijven gebruiken Javascript?

Javascript wordt ondersteund door vrijwel elke browser. Veel grote bedrijven gebruiken daarom ook javascript. Dit is niet meer weg te denken uit de huidige wereld. Een aantal van dit soort grote bedrijven zijn: Google, Microsoft, Apple & Amazon.
