Wat is HTML?


HTML staat voluit voor Hyper Text Markup Language en kan gezien worden als een taal om een pagina op te maken. Eigenlijk is HTML dus niet echt een programmeertaal, het is eerder een opmaaktaal. Met HTML kan je namelijk bepalen welk element waar op je webpagina verschijnt, voor een geordende pagina is HTML dus eigenlijk onmisbaar.

Meer weten over html? Klik hier.

Wat is CSS?


CSS staat voor Cascading Style Sheets en is net als HTML eigenlijk ook niet echt een programmeertaal. CSS kan eerder gezien worden als een ‘stijlbladtaal’. Wat houdt een stijlbladtaal dan precies in? Dit betekent dat je door CSS te gebruiken toestaat dat bepaalde stijlelementen worden toegepast op geselecteerde delen van je html webpagina.
Meer weten over css? Klik hier.

Wat doen wij op de opleiding met HTML / CSS?


Op de opleiding software development hier op het technovium maken wij veel gebruik van HTML en CSS. Dit is bijvoorbeeld om kleine projecten / websites te bouwen. In elke periode krijg je een eind project, dit is tevens ook je cijfer voor dat vak. In de meeste gevallen is dit een volledige website bouwen en dit wordt dus gedaan met HTML & CSS
