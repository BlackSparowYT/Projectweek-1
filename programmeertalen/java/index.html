Java

Wat is Java?

Java is een programmeertaal en computingplatform dat voor het eerst werd uitgegeven in 1995 door Sun Microsystems. Java heeft zich ontwikkeld van een bescheiden platform tot een betrouwbaar platform dat van vandaag de dag overal in de digitale wereld wordt gebruikt voor het bouwen van services en applicaties. Ook voor nieuwe, innovatieve producten en digitale services die voor de toekomst worden ontworpen, wordt gebruikgemaakt van Java.

Wat doen wij met Java?

Hier op deze opleiding maken wij ook gebruik van Java. Dit is om projecten te bouwen. Een voorbeeld hiervan is een kleine game. Dit gaat allemaal gepaard met alle deelstappen die je maakt voor je daadwerkelijk code gaat typen. Je leert hier alles wat je nodig hebt om een goede kennis over Java te bemachtigen.

Welke bedrijven gebruiken java?

Er zijn niet echt bedrijven die zich specifiek richten op Java. Wel zijn er meerdere dingen gemaakt met Java en gebruiken ook veel mensen ook Java. voorbeelden van games die gemaakt zijn met Java zijn: Minecraft, God of war, Asphalt 2 & Old school RuneScape.
