SQL

Wat is SQL?

SQL is een ANSI/ISO-standaardtaal voor een relationeel databasemanagementsysteem. Het is een gestandaardiseerde taal die gebruikt kan worden voor taken zoals het bevragen en het aanpassen van gegevens in een relationele database. SQL kan met vrijwel alle moderne relationele databaseproducten worden gebruikt.

SQL wordt door veel bedrijven gebruikt. Dit is omdat een database op vrijwel elke (grote) website runt. Dit gaat dan dus vooral ook om grote bedrijven.
