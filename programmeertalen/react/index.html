React

Wat is React?

React is een Javascriptbibliotheek om gebruikersinterfaces mee te bouwen. React werd in 2011 door Meta ontwikkeld vanuit de behoefte om de code van grote webapplicaties beter beheersbaar te maken. Het was in eerste instantie een intern project en werd gebruikt voor het Facebook- en Instagramplatform.
Meer weten over React? Klik hier.

Wat doen wij met React?

React is een onderdeel wat je krijgt als je kans maakt op het doorstromen. Sommige jongens hebben al ervaring met programmeren voordat ze op de opleiding komen en daardoor kan het niet uitdagend genoeg zijn. Hier komt react dan in het leven geroepen hier.

Welke bedrijven gebruiken React?

Ook react is inmiddels niet meer weg te denken. Zoals je al eerder las is react als eerste instantie bedoeld voor een intern project en werd het gebruikt voor facebook en instagram. Inmiddels gebruiken meerdere grote bedrijven react. Voorbeelden hiervan zijn: Facebook, instagram, Netflix, Reddit, AirBnB en Uber.
